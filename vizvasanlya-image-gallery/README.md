Organize visual assets, albums, tags, exports, and campaign-ready media collections.

## Features

- **Asset Management** — Upload, organize, and tag visual content
- **Album Organization** — Group assets into collections and campaigns
- **Tag System** — Categorize and filter by tags and metadata
- **Export Ready** — Track export status and format readiness
- **Search & Filter** — Find assets by tag, status, or type
- **Analytics View** — Asset distribution and usage insights
- **Import / Export** — JSON-based gallery data portability

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES2022) |
| Storage | LocalStorage |
| Server | Python HTTP server (dev) |

## Getting Started

### Prerequisites

- Python 3.x (for dev server)
- Modern web browser

### Installation

```bash
git clone https://github.com/vizvasanlya/First.git
cd First/vizvasanlya-image-gallery
```

### Run

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173)

### Docker

```bash
docker compose up --build
```
